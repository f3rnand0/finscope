# Issue Tracker

No issue tracker is active for this repo.

## Policy

- Do not create GitHub issues for this repo.
- Do not create GitLab, Jira, Linear, or other external tickets.
- Do not create local markdown issues under `.scratch/`.
- Do not treat PRDs, plans, or triage notes as published work items unless the user explicitly gives a destination.

## When a skill says "publish to the issue tracker"

Ask the user where the item should be published before writing anything. Acceptable future destinations might include GitHub Issues, local markdown files, or another tracker, but none is configured now.

## When a skill says "fetch the relevant ticket"

Ask the user for the source of truth. There is no configured issue location to inspect automatically.
